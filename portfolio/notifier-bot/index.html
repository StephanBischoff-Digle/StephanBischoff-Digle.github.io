Magazino - Notifier Bot

Description

The Magazino robots (TORU, SOTO) had to communicate errors somehow; especially during performance evaluations. For this purpose the robot had LEDs in front and in the back, such that errors could be signaled visually from afar. Additionally, operators connected to the robot via ssh were able to collect more detailed information over a tmux session dedicated to logging the robots state.

This kind of informationis useful as soon as it is generated and to more than just the person connected to the machine (which might be zero). The Notifier Bot checked the state of the robot in configured intervals and send them via specified channels (a chat channel or email, for example).
