magazino/Warehouse Creator

The Project

Description

The Magazino robots (TORU, SOTO) can generate a map using two dimensional LIDAR. With this map, a robot can localize itself and find paths from one to another location. However, for the autonomous operation of the robot, it needs more information. This semantic data (like compartment names, locked off areas and similar) can not easily be generated by the robot and therefore has to be modeled and stored in a configuration file.

The Warehouse Creator is the graphical front end to that configuration file. With it shelves can be aligned with the robots LIDAR map, any warehouse object can be named and areas can be allocated and configured.

Although the project had a potentially high value for the deployment of the robots, it was without maintenance for a while when I took it over as a whole. The main functionality was already present: the GUI, the ability to create the warehouse model and the alignment of it with the LIDAR map, the import and export of the configuration file.
